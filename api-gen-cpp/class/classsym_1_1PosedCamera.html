Class sym::PosedCamera¶

template<typename CameraCalType>
class sym::PosedCamera : public sym::Camera<CameraCalType>¶

Camera with a given pose, camera calibration, and an optionally specified image size. If the image size is specified, we use it to check whether pixels (either given or computed by projection of 3D points into the image frame) are in the image frame and thus valid/invalid.

Computes a point written in the global frame along the ray passing through the center of the given pixel. The point is positioned at a given range along the ray.

Args: pixel: Vector in pixels in camera image. range_to_point: Distance of the returned point along the ray passing through pixel epsilon: Small value intended to prevent division by 0.

Return: global_point: The point in the global frame. is_valid: 1 if point is valid
